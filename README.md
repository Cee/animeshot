
AnimeShot
=========

An open and searchable image repository in Node.js (WIP, but [live](https://as.bitinn.net/))


## Features

- [x] Progressive upload
- [x] Drag-n-drop upload
- [x] Image resize
- [x] Searchable text and source
- [ ] OAuth
- [ ] Moderation
- [ ] Rating
- [ ] User collections
